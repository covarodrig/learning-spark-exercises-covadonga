# Libro Learning Spark - Ejercicios

Repositorio con los ejercicios del libro "Learning Spark" para la formación.

## Capítulos Cubiertos

- **Capítulo 2**: Getting Started with Apache Spark
- **Capítulo 3**: Apache Spark's Structured APIs
- **Capítulo 4**: Spark SQL and DataFrames
- **Capítulo 5**: Interacting with External Data Sources
- **Capítulo 6**: Datasets
- **Capítulo 7**: Optimizing and Tuning Spark Applications
- **Capítulo 9**: Building Reliable Data Lakes

## Estructura del Proyecto
```
src/main/scala/
├── chapter02/  # Introducción y primeros pasos
├── chapter03/  # APIs estructuradas
├── chapter04/  # Spark SQL básico
├── chapter05/  # Fuentes de datos externas
├── chapter06/  # Datasets tipados
├── chapter07/  # Optimización
└── chapter09/  # Data Lakes
```

## Cómo ejecutar
```bash
# Compilar el proyecto
sbt compile

# Ejecutar un ejercicio específico
sbt "runMain chapter02.Exercise01_FirstSparkApp"
```

## Requisitos

- Scala 2.12.x
- Apache Spark 3.3.x
- SBT 1.x
- Java 8 o 11
